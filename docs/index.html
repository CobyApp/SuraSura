🎙 リアルタイム音声翻訳

言葉の壁をなくそう

話すだけで即座に翻訳。旅行・ビジネス・語学学習、あらゆるシーンで活躍するリアルタイム翻訳アプリ。

App Store でダウンロード

主な機能

🎙

リアルタイム音声認識

話した内容をその場で文字に変換。高精度な音声認識エンジンを搭載。

⚡️

即時翻訳

認識したテキストを瞬時に翻訳。タイムラグなしでスムーズなコミュニケーション。

🤝

フェイス・トゥ・フェイスモード

画面を向かい合わせにするだけで自然な双方向会話。互いの言語で話せます。

🔊

自動読み上げ（TTS）

翻訳完了と同時に自動で音声再生。画面を見なくても翻訳結果がわかります。

🌙

ダーク / ライトモード

シーンに合わせてモードを切り替え。目に優しいダークモードにも対応。

🌐

多言語対応

世界中の主要言語をサポート。旅行先でも安心してご利用いただけます。
